- ⚠️ **Важно**: Для включения режима отладки замкнуть контакт 3 и GND (Rele, receiver)

## 📨 Протоколы обмена данными

### 🎮 Remote Controller → Receiver
#### • Передача команды с пульта
```json
{
    "command": "command_name",
    "alarm": "fire",
    "typeAlarm": "traning",
    "code": "123456"
}
```



### 🌐 Receiver → Server
#### • Передача команды с приёмника
```json
{
  "type": "remoteCommand",
  "receiver_id": "1",
  "data": {
    "command": "command_name",
    "alarm": "fire",
    "typeAlarm": "traning",
    "code": "123456"
  }
}
```

#### • Ping-ответ

```json
{ 
    "type": 'ping' 
}
```

#### • Регистрация устройства

```json
{ 
    "type": 'register', 
    "nameDevice": nameDevice 
    }
```
